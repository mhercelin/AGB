Consignes de mise à jour du script AlexGuestbook
de la version 5.0.* à la version 5.0.2


Cette mise à jour au livre d'or AlexGuestbook est facultative. Elle corrige quelques bugs et erreurs. Elle ajoute également deux nouvelles langues (Perse et Turque).
La liste des modifications et corrections apportées au script est disponible dans le fichier changelog_fr.txt.
Retrouvez toutes les informations, les mises à jour et l'assistance du script sur www.alexguestbook.net


Avant toutes modifications, faites une sauvegarde de la base de données (voir l'interface d'administration du script) et de vos skins si vous les avez personnalisés.


  1. Avant de commencer, assurez-vous d'avoir téléchargé l'archive ZIP (v5x-v502.zip) contenant les fichiers de mises à jour. Ce fichier ZIP permet de mettre à jour aussi bien une version 5.0 ou 5.0.1 vers la v5.0.2.

  2. Les skins (ou thèmes) n'ont été que très peu modifiés (à l'exception du skin double_NewLook dont les corrections sont données plus bas) mais nous vous conseillons tout de même de les metttre à jour si vous ne les avez pas personnalisés. Dans le cas contraire, la mise à jour n'est pas obligatoire (car la sécurité n'est pas impactée).
    Vous pouvez obtenir la liste des modifications de chaque fichier avec un logiciel de comparaison de fichiers (tel WinMerge).

  3. La seule étape de cette mise à jour consiste donc à envoyer (avec votre logiciel FTP) les fichiers du dossier de l'archive correspondant à votre version actuelle. Si votre logiciel vous demande d'écraser certains fichiers, confirmez l'opération.

Liste des étapes de mise à jour du skin double_NewLook :

Nous rappelons qu'il est fortement conseillé de prendre le skin double_NewLook fournit avec la dernière version (donc à jour) et de le personnaliser plutôt que de mettre à jour un skin personnalisé.

Avant de mettre à jour votre skin, celui-ci doit être en v5. Si ce n'est pas fait, voyez les étapes de mise à jour dans le fichier Instructions pour le passage de la version 4.x à la version 5.0.2.
